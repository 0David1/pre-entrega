TechStore

Incluye secciones de productos, reseñas y contacto, con un diseño moderno y responsivo.

Estructura

index.html → Página principal con video y tabla de mensajes.

/pages/productos.html → Tarjetas de productos creadas con Flexbox.

/pages/reseñas.html → Reseñas organizadas con CSS Grid.

/pages/contacto.html → Formulario de contacto con Formspree.

/css/ → Archivos de estilos separados por secciones.

/img/ → Logos e imágenes de productos.

Tecnologías

HTML5

CSS3 (Flexbox, Grid, Media Queries)

Font Awesome

Google Fonts
