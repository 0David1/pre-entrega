TechStore

Incluye secciones de productos, reseñas y contacto, con un diseño responsivo y un carrito funcional usando JavaScript.

Estructura

index.html → Página principal con video/banner.
/pages/productos.html → Tarjetas de productos creadas con Flexbox, cargadas desde una API.
/pages/reseñas.html → Reseñas organizadas con CSS Grid.
/pages/contacto.html → Formulario de contacto con Formspree.

/css/ → Archivos de estilos separados por secciones.
/js/ → Módulos JS (carrito, UI, fetch de productos).
/img/ → Logos e imágenes del sitio.

Tecnologías

HTML5

CSS3

JavaScript

Font Awesome

Google Fonts

FakeStoreAPI para productos

Formspree para el formulario de contacto
